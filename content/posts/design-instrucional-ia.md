# IA no Design Instrucional: ferramenta ou substituto?

Já escrevi centenas de aulas. Desde apostilas para concursos públicos até conteúdos de pós-graduação. E nos últimos dois anos, uma ferramenta entrou para o meu fluxo de trabalho de forma silenciosa e irreversível: os grandes modelos de linguagem.

A pergunta que mais ouço nos grupos de professores é: **"Isso vai substituir o designer instrucional?"**

Minha resposta curta: não. Mas vai transformar o que significa ser um.

## O Que a IA Faz Bem

Quando uso um LLM para criar conteúdo educacional, ele se sai bem em tarefas de **estruturação**:

- Gerar um esboço de unidade a partir de ementa
- Criar questões objetivas sobre um tema definido
- Adaptar linguagem técnica para diferentes perfis de alunos
- Sugerir exemplos práticos para conceitos abstratos

```
Prompt que funciona:
"Você é um designer instrucional sênior. 
Preciso criar 3 atividades reflexivas sobre 
[tema X] para alunos de pós-graduação em gestão. 
Cada atividade deve ter: objetivo claro, 
enunciado, critério de avaliação formativa."
```

O output é um ótimo ponto de partida. **Nunca é o produto final.**

## O Que a IA Não Faz

A IA não sabe:

- **O perfil real do seu aluno** — quem é ele, o que sabe, o que teme
- **O contexto institucional** — restrições de plataforma, cultura da turma, ritmo do curso
- **A intencionalidade pedagógica** — por que essa sequência? Por que esse exemplo e não outro?

Um conteúdo gerado por IA sem revisão crítica é como uma planta baixa sem o arquiteto: tecnicamente possível, mas ignorando tudo o que importa.

## Meu Fluxo Atual

Hoje, para criar uma unidade de conteúdo, eu:

1. **Defino a intenção pedagógica** — o que o aluno deve ser capaz de fazer depois?
2. **Peço ao modelo um rascunho** com base na ementa e objetivos
3. **Reviso brutalmente** — reescrevo exemplos, ajusto linguagem, elimino generalismos
4. **Adiciono âncoras reais** — casos do mercado, reflexões próprias, contexto local
5. **Valido com par** — um colega lê antes de publicar

O LLM entrou no passo 2. Isso economizou horas. E liberou energia para o que realmente diferencia um bom conteúdo: a curadoria e a voz.

## Conclusão

A IA não vai substituir o designer instrucional que pensa. Vai engolir o que só executa. E honestamente — já era tempo.

O professor que usava a IA em 2023 está produzindo mais e melhor. O que resistiu por princípio está atrasado. A questão não é se você vai usar. É se vai usar com intenção.
